# 🚀 Code Debug Agent

An AI-powered debugging assistant that analyzes and fixes code errors across multiple programming languages.

## 🔧 Features
- Detects syntax & runtime errors
- Suggests fixes using AI (Groq + LangChain)
- Supports multiple languages
- Simple Streamlit UI

## 🛠 Tech Stack
- Python
- Streamlit
- LangChain
- Groq API
